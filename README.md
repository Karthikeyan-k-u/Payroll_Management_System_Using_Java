# Payroll_Management_System_Using_Java

This Java program implements a simple Payroll Management System to calculate and display an employeeâ€™s salary details. It uses a class to store employee information such as ID, name, department, and basic salary. The program calculates gross salary by adding allowances and net salary by deducting taxes. It demonstrates the use of constructors, methods, encapsulation, calculations, formatted output, and user input handling using the Scanner class.

**Algorithm**

*Display the payroll system title.
*Read employee ID, name, department, and basic salary.
*Create an Employee object using the constructor.
*Calculate gross salary by adding allowances to basic salary.
*Calculate net salary by deducting taxes from gross salary.
*Display the employee pay slip with all details.
*Close the input stream.
